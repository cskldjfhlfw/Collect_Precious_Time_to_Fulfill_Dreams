## Excel导入模板建议

### 必填字段标记 (*)

#### papers (论文)
- title * (标题)
- status * (状态: draft/submitted/accepted/published)

#### patents (专利)
- name * (名称)
- patent_number * (专利号)
- patent_type * (类型: invention/utility/design)

#### projects (项目)
- name * (名称)
- project_number * (项目编号)
- project_type * (类型: national/provincial/enterprise)

#### software_copyrights (软著)
- name * (名称)
- registration_number * (登记号)

#### competitions (竞赛)
- name * (名称)
- level * (级别: international/national/provincial/school)

#### conferences (会议)
- name * (名称)

#### cooperations (合作)
- organization * (机构名称)

## 字段说明补充

### JSONB字段格式

**authors, inventors, developers, team_members, participants** - JSON数组格式:
```json
[{"name": "张三", "affiliation": "XX大学"}]
```

**related_projects** - JSON对象格式:
```json
{"project_id": "uuid", "project_name": "项目名称"}
```

**filters** - JSON对象格式:
```json
{"status": ["published"], "date_range": {"start": "2024-01-01"}}
```

### ARRAY字段格式

**keywords, tags** - 逗号分隔或数组格式:
```
机器学习,深度学习,人工智能
或
["机器学习", "深度学习", "人工智能"]
```

### 日期格式

所有日期字段统一使用: `YYYY-MM-DD`  
示例: `2024-11-15`

### 时间戳格式

created_at, updated_at: `YYYY-MM-DD HH:MM:SS`  
示例: `2024-11-15 20:30:00`
